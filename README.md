# test_requests
# Программа для тестирования доступности серверов по HTTP протоколу
Программа замеряет время выполнения запроса и выводит итоговую статистику по скорости ответа сервера.<br>
Программа принимает следующие аргументы:<br>
•	-H со значением хостов (указывать через запятую без пробелов), на которые будут выполняться запросы или -F – путь до файла (в формате .txt) со списком адресов, разбитый на строки. Одновременно может быть указан только один из ключей.<br>
•	-С со значением количества запросов (по умолчанию 1)<br>
•	-O – путь до файла (в формате .txt), куда нужно сохранить вывод (по умолчанию вывод отправляется в консоль)<br>
Пример работы программы:<br>
Ввод в командную строку: python requests_test.py -H https://yandex.ru,https://google.com,https://yaex.ru -С 5<br>
Вывод будет производиться в консоль.<br>
